# E-Commerce Big Data Analytics & Recommendation System
Github Repo Link :- https://github.com/IshikaThilomi/Ecommerce-Bigdata-Recommendataion-System-and-Apache-Spark-Analysis
Data Set is from Kaggale : https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

## Project Overview
This mini-project demonstrates the application of Big Data Analytics and Recommendation Systems using Apache Spark (PySpark) on the Brazilian E-Commerce Public Dataset by Olist. The project fulfills both Part A and Part B of the assignment guidelines:
* **Part A (Big Data Analytics):** Data cleaning, feature engineering, and exploratory data analysis (EDA) to extract insights regarding revenue, order volume trends, and logistics performance.
* **Part B (Recommendation System):** A Collaborative Filtering recommendation engine built using Spark MLlib's Alternating Least Squares (ALS) algorithm to suggest relevant products to users based on historical review scores.
